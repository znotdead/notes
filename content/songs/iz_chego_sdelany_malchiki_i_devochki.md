### Из чего сделаны мальчики и девочки

Из чего только сделаны мальчики?
Из чего только сделаны мальчики?
Из колючек, ракушек
И зеленых лягушек,
Вот из этого сделаны мальчики.

Из чего только сделаны девочки?
Из чего только сделаны девочки?
Из конфет и пирожных,
Из сластей всевозможных,
Вот из этого сделаны девочки.
